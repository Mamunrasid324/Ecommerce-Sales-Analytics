# 🛍️ Retail Sales Analysis using Python

## 📌 Project Overview

This project analyzes retail sales data using Python and Pandas. It simulates a real-world retail business by generating customer, product, and sales datasets, then performs data cleaning, exploratory data analysis (EDA), KPI calculation, and data visualization.

The goal of this project is to demonstrate practical Data Analyst skills using Python.

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
